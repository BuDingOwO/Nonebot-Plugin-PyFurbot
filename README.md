## Install

```dos
git https://github.com/BuDingOwO/Nonebot-Plugin-PyFur.git
```
## Nonebot 版本的 Furbot(绒狸)

需要像绒狸官方申请Token

基于 绒狸开源版 [Furbot - Mirai](https://github.com/furleywolf/Furbot-Mirai)

目前支持的功能

1. 来只毛
2. 来只 xxx
3. 找毛图
4. 每日鉴毛

配置文件路径:

```
<bot.py>/data/PyFurBot/config.json
```
